# RMC
Code for "Leave-One-Out Analysis for Nonconvex Robust Matrix Completion with General Thresholding Functions"
- **PROPACK package**: Downloaded from https://candes.su.domains/software/svt/code.html (originally available at http://sun.stanford.edu/~rmunk/PROPACK/).
- **compute_X_Omega.mexw64**: Obtained from https://pages.cs.wisc.edu/~yudongchen/rpca_gd/.
- **test_RMC.m**: Script for testing RMC-SOFT and RMC-SCAD on synthetic examples. It can also be used on videos from the VIRAT dataset after processing as described in the paper.
